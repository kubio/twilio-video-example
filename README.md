# twilio-video-example

# Dependency
- NodeJS(v8.12 or later)
- yarn
- Twilio Account

# Setup
1. create .env
```
$ cp env.example .env
$ vi .env 
```
create twilio account & setup Programmable Video.
Please set Sid/secret to .env.


2. run
```
$ cd client && yarn install
$ cd .. && yarn start
```

# Usage
1. access http://localhost:3001/
2. enter name
3. join room
